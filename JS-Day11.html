Array Methods

  1. push()
    • This method is used to add an element to the end of the array. This method changes the length of the array.
  2. unshift()
    • The unshift() method is used to add a new item as the first element of the array.
  3. pop()
    • This method is used to remove and return the last element of the array.
  4. shift()
    • The shift() method is used to remove and return the first element of the array.
  5. splice()
    • The splice() method is used to remove or add elements in the array by specifying the index.
