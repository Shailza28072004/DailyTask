Loops in JavaScript

  1. While Loop:
      The while loop loops through a block of code as long as a specified condition is true. 
                          
                        Syntax-:
                            while(condition){
                                //code to execute
                            }
  2. Do-While Loop:
      The do...while statements combo defines a code block to be executed once, and repeated as long as a condition is true. The do...while is used when you want to run a code block at least one time. 
                      
                    Syntax-:
                    do{
                        //code
                    }
                    while(condition);
  3. for Loop:
      Loops can execute a block of code a number of times. 
                          
                        Syntax-:
                        for(inilization;condition;inc/dec){
                            //code
                        }
  4. Nested Loops:
      The nested loops means one loop inside another.
