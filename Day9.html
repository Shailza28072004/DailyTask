What are Links?

link is a clickable text image or the graphic that navigate user to any spefic location when clicked

What are hyperlinks?

The hyperlink is a clickable text,graphic,or image that navigate over a specified HTTP

Navigation means to move from one place to another

Types of links in HTML->

There are two types of links in HTML

  1. Intradocument links
    • The Intradocument links are used to move within webpage
  2. Interdocument links
    • The Interdocument links are used to move from one webpage to another

Anchor Tag

The anchor tag is used to create the hyperlinks in HTML

We can achieve this or you can say that we can make a Intradocument hyperlink by using the anchor tag
