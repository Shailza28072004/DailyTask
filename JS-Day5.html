conditional Statements

The conditional Statements are used when we want to perform some action on the basis of the condition. for example if somebody want to check that if a number is even or not then the Consitional Statements come into use

In JavaScript we have the following statement to check for the conditions:

  1. If statement
  2. If-Else statement
  3. If-else-if-else statement
  4. Ternary Operators
  5. Switch statement
Lets understand about the syntax of above Statements 
            1.If statement
            The syntax of if statement is:
              if(condition)
              {
                  //code
              }

            2.If-Else statement
            The syntax of if statement is:
              if(condition){
                  //code
              }
              else{
                //code
              }

              3.If-else-if-else statement
              The syntax of if statement is:
                if(condition){
                    //code
                }
                else if(condition){
                  //code
                }
                else{
                    //code
                }

                4.Ternary Operators
                  condition?works if condition is true:works if condition is false
