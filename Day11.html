String Methods

We are discusing about the various string methods in JavaScript

  1. charAt()
    • The charAt() method returns the character at a specified index (position)
  2. indexOf()
    • Returns the index of the first occurrence of searchValue in the string. Returns -1 if searchValue is not found.
  3. lastIndexOf()
    • Returns the index of the last occurrence of searchValue in the string. Returns -1 if searchValue is not found.
  4. trim()
    • Removes whitespace from both ends of a string and returns a new string. It does not affect whitespace within the string.
  5. substring()
    • Returns a part of the string between start and end indices. The start index is inclusive, while the end index is exclusive.
  6. Split()
    • Splits a string into an array of substrings based on a specified separator.
  7. slice
    • Extracts a section of the string and returns it as a new string, without modifying the original string. The start index is inclusive, and the end index is exclusive.
